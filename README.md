## Быстрый старт

> Требования: **Python 3.9+**. Рекомендуется виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
playwright install chromium
```

На Windows: `.\.venv\Scripts\activate`

### Запустить все тесты

```bash
pytest test_ui.py -v
```

**Ожидаемый результат:** `24 passed, 7 failed` (31 тест всего)

7 тестов падают из-за **известных багов** стенда модерации (**BUG-UI-013 … BUG-UI-019**) — в тексте `assert` указан ID бага и краткое описание. Баги по скриншоту Avito — **BUG-UI-001 … BUG-UI-012** (автотестами не покрываются).

Исправить само приложение на Netlify из этого репозитория нельзя: в нём только тесты и отчёты; исходный код стенда в задачу не входит.

**Стенд — SPA:** прямой переход по URL вида `https://…/stats` в новой вкладке может отдать страницу 404 Netlify; в сценариях и в `test_ui.py` переход на статистику выполняется через пункт меню «Статистика» после загрузки главной.

```bash
pytest test_ui.py -v --headed
pytest test_ui.py -v -k "ThemeToggle"
pytest test_ui.py -v -k "not ThemeToggle"
```
