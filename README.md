## Быстрый старт

> Требования: **Python 3.9+**

```bash
pip install -r requirements.txt
playwright install chromium
```

### Запустить все тесты

```bash
pytest test_ui.py -v
```

**Ожидаемый результат:** `24 passed, 4 failed`

4 теста падают из-за **известных багов** приложения (BUG-UI-007, BUG-UI-008, BUG-UI-009, BUG-UI-018) —
сообщение об ошибке содержит ID бага и описание причины.

```bash
pytest test_ui.py -v --headed             # с отображением браузера
pytest test_ui.py -v -k "ThemeToggle"     # только мобильные тесты
pytest test_ui.py -v -k "not ThemeToggle" # только десктопные тесты
```
